# Tic-Tac-Toe Game

This is a simple implementation of the classic Tic-Tac-Toe game using HTML, CSS, and JavaScript.

## Screenshots
![Dashboard Screenshot](https://github.com/shlokkatu/TicTacToe/blob/d1ad4b2b98b9f59cf12547d40177325099ae8d4e/Content.png)


## Features

- Play Tic-Tac-Toe against another player.
- Automatically detects wins, draws, and updates the game status.
- Responsive design, suitable for desktop and mobile devices.

## How to Play

1. **Setup**:
   - Clone the repository to your local machine.

2. **Run**:
   - Open `index.html` in a web browser.

3. **Gameplay**:
   - Click on any empty cell to place your symbol (X or O).
   - Players take turns to make a move.
   - The game ends when a player wins or if it's a draw.

## Technologies Used

- HTML5
- CSS3 (Flexbox for layout)
- JavaScript (DOM manipulation for game logic)
